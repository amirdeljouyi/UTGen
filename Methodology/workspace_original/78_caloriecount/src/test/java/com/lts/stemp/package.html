A very simple text template processing package.

This package is intended to transform a text template file into a text output file. The package it typically used to generate code from a text template.

The system works by first loading a template file: 

    ...
    String fname = "some file name";
    Template template = Template.load(fname);

    ...

    Map data = <method to generate some data>
    String outfile = "some file name";
    template.generate(outfile, data);
